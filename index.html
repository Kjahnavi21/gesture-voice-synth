Gesture Voice Explorer

This small prototype maps simple user input to voice-like sound using the Web Audio API. Move inside the gesture area to control the sound, or use the sliders below for direct control.

Move your mouse or trackpad here

Left/right changes pitch. Up/down changes volume.

Pitch: 220 Hz

Volume: 0.10

Input mode: Not started

Fine-Tune Controls

These sliders provide a direct way to adjust the same sound parameters controlled by the gesture area.
