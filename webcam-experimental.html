Webcam Gesture Sound Experiment

This optional experiment uses webcam input to control sound in the browser. It compares each camera frame to the previous frame, estimates where movement is happening, and maps that movement position to audio.

Movement left/right changes pitch. Movement up/down changes volume.

Camera Input

Try moving your hand or face in different parts of the camera frame.

Live Sound Values

Pitch: 220 Hz

Volume: 0.01

The displayed values are smoothed so the sound does not jump too sharply.
